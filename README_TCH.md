# 異世界創造者存檔解密加密工具
一個幫助你加密解密異世界創造者存檔的工具。

# 如何使用
> 注意：使用前請先備份存檔！

* 修改 decode(解密) 的腳本的檔案路徑
* 跑 decode 腳本，可以取得一個明文的 json 檔
* 更改 json 檔，修改 encode(加密) 腳本的檔案路徑
* Encode 腳本會依據 json 檔的內容生成一個存檔
* 備份好舊的存檔。將修改好的存檔覆蓋到存檔的地方
